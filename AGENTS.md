# Agent Instructions

When working in this repository, use the Codex workspace docs as the source of truth.

- Primary config path: `.codex`
- Primary rules path: `.codex/rules`

Execution requirements:

1. Read `AGENTS.md`, then load and follow `.codex/rules/index.md`.
2. Load the specific rule files linked from `.codex/rules/index.md` that match the area you are changing.
3. Treat compatibility files in other tool directories as shims only, not as separate sources of truth.
